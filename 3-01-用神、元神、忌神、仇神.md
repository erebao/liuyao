## 用神、元神、忌神、仇神

用神，是指断卦中的主事爻，又称用爻，简单来说就是求测什么，就相应以卦中哪一种六亲为用神，如测财运就以卦中的妻财爻为用神。

各六亲的含义大致归纳如下：
```
1、父母。父母的定义不仅仅是指父母亲，凡生我，对我有宜，能够庇护、保护我的都叫父母。
    - 代表父母、长辈、老人、老师、师傅等人际关系
    - 代表房屋、衣服、雨伞等能庇护我的物品
    - 代表书信、文书、契约、合同、成绩等与文星有关的事物
    - 代表信息、消息、短信等现代通讯
    - 代表车辆、飞机等运输工具
    - 代表下雨、下雪等天气现象 
2、子孙。受我庇护的人或物
    - 代表儿女、孙子、孙女、幼儿、胎儿、晚辈、徒弟、下属等人际关系
    - 代表医生、医药、救治、医疗器械、健身器材等与救死扶伤有关的事物
    - 代表忠臣、良将、神灵等正派角色
    - 代表六畜、六禽等动物，凡是测动物都已子孙用神
    - 代表娱乐、开心、玩乐、消遣等高兴快乐的事物
    - 代表僧侣、道士、天师等克鬼职业
    - 代表顺风、福德、天晴等有宜之事
3、兄弟。同辈以及相互竞争的人或事物
    - 代表兄弟姐妹、堂兄堂妹、关系较好的表兄表妹、关系较好的朋友、同事、同僚、师兄弟等同辈的人
    - 代表假肢、假牙、假发等替代品
    - 代表同行等竞争对手
    - 代表破耗、截财、花费等破财之事
    - 代表阻碍、阻拦、门槛等不顺畅之事
    - 代表大风、逆风等天气现象
4、妻财。受我掌控和支配的人或事物
    - 代表妻子、情人、女友、暧昧对象、相亲对象、奴仆等女性
    - 代表钱、金银、首饰、报酬、薪水、奖金等与钱财有关的物品
    - 代表私人用品，失物等
    - 代表粮食、食物、酒水等饮食
5、官鬼。克我的，制约我的人或事物
    - 代表官员、上司、老板、领导、丈夫、男友、情郎等管束我的人
    - 代表盗贼、犯罪分子等不正当的职业
    - 代表疾病、灾祸等凶险之事
    - 代表事业、官运、工作之事
    - 代表疑惑、焦虑、担心、害怕等忧患之事
    - 代表鬼怪、鬼神、尸体、去世的人等
6、世爻与应爻。测自身都以世爻为用神，应爻为对方，一般测和自己不想干的人取应爻为用神。
    - 用神：如测长辈身体情况，就以父母为用神，男测感情，就以妻财为用神，测猫走失，就以子孙为用神，其余以此类推
    - 元神和忌神：凡是生用神的叫做元神，在预测时多为有利因素，也代表人的思想、想法、思维、意识等，克用神的叫做忌神，在预测时多为阻碍，灾煞、死对头等
    - 仇神：克元神的叫做仇神，通过克害元神来间接对用神产生不好的影响，预测时多表现为仇人、小人、背后说坏话之人等
```

例如：求财测得离为火卦，应该以妻财酉金为用神，土生金，那么卦中凡属土的就是妻财的元神，火克金，那么卦中凡属火的就是妻财的忌神，土为元神，木克土，那么卦中凡是属木的就叫做仇神
```
        离为火
兄弟巳火 ————世 (忌神)
子孙未土 —— ——  (元神)
妻财酉金 ————   (用神)
官鬼亥水 ————应
子孙丑土 —— ——  (元神)
父母卯木 ————   (仇神)
```
