## 月建

月建，又叫月将、月令。是指起卦时月份的天干地支组合，预测时以地支为主，天干一般不考虑。

卦中之爻根据月建与爻的互相关系，大致可以分为十个层次：

```
1）临月建
    卦中之爻与月建地支完全相同称为临月建。
    断卦时该爻犹如有金刚护体，即使受到他爻克害也可以在当月内无妨，被该爻克害之爻则受伤极深难以抵挡，如无其他救助，在当月内即见凶危。
2）月建生合
    卦中之爻得月建相生又相合，共有三种：午(5)月未土爻、辰(3)月酉金爻、亥(10)月寅木爻。
    即使受克，在月内也不会显衰败，而被该爻所克之爻则受伤极深，如无其他救助，多在三个月内应凶。
3）月扶
    卦中之爻的地支五行与月建五行相同称为月扶。如：寅(1)月卯木爻、申(7)月酉金爻等等...
    即使受克，在月内也可以抵挡不显衰败。
4）月生
    卦中之爻受月建相生，如未月申(7)金、酉金爻。
    即使受克，在月内也可以抵挡而不显凶危。
```
以上四类中，临月建、月建生合、月扶称为旺，月生称为相，统称为旺相，都属于有用之爻。
```
5）月建平合
    卦中之爻得到月建相合但是不相生，共有六种：寅(1)月亥水爻、酉(8)月辰土爻、未(6)月午火爻、戍(9)月卯木爻、申(7)月巳火爻、子(11)曰丑土爻。
    逢克依然受伤，只是程度略轻，只要得到帮扶即可解救。
6）月气
    指每个季度最后一个月，共有四种（辰月寅卯木有气、未月巳午火有气）：
        辰月是春季最后一个月，寅木、卯木有气；
        未月是夏季最后一个月，巳火、午火有气；
        戍月是秋季最后一个月，申金酉金有气（但是由于戍土本来就生金，预测时一般论月建相生，属于旺相）；
        丑冬是冬季最后一个月，亥水、子水有气（由于土克水，预测时一般论衰）；
    受克可以略微减轻伤害，但是受伤程度较深。
```
以上两类属于有气的范围，爻的旺衰程度已经大不如前面四类，但尚有部分力量
```
7）休囚
    指月建对该爻没有任何做用，该爻生月建或者克月建。如：月建为丑土，卦爻为寅木，此时丑土对寅木没有任何帮助。
    受克即伤，没有任何抵挡。
8）月建克合
    卦中之爻受月建相克，同时又与月建相合，共有三种：卯(2)月戍爻、巳(4)月申爻、丑(12)月子爻。
    此爻犹如与皇帝意见不合而被处理之人，显得比较衰败，但在处理时仍然会念及旧情，会有所保留，网开一面。
9）月克
    指卦中之爻被月建相克，如寅(1)月丑辰戍未巳爻，都被月建所克。
    当月内衰败非常，命在旦夕，十分危险，如没有其他力量相生相救，一般在月内即见凶危。
10）月破
    指卦中之爻地支上与月建相冲。如：子(11)月午爻，午(5)月子爻，寅(1)月申爻，申(7)月寅爻，辰(3)月戍爻，丑(12)月未爻等。
    属于衰败彻底之象，即使有其他力量救助，仍然为衰。除特殊情况外，在月内，月破之爻逢生无用，救之不起，属于最为衰败之爻。
    月破之爻是不分五行生克的，只要与月建相冲，即为月破。
```
以上四类属于衰的范围，在吉凶上属于无用之爻，若无其他条件解救，多应凶危。

#### 大致情况归纳如下：
```
寅月，寅木临月建，卯木旺，巳午火相，亥水平合有气，子水、酉金休囚，辰戍丑未土衰，申金月破。
卯月，卯木临月建，寅木旺，巳午火相，亥子水、申金休囚，戍土克合衰，辰丑未土衰，酉金月破。
辰月，辰土临月建，酉金生合旺，丑未土旺，申金相，寅卯木有余气，巳午火休囚，亥子水衰，戍土月破。
巳月，巳火临月建，午火旺，辰戍丑未土相，子水、寅卯木为休囚，申金克合衰，酉金衰，亥水月破。
午月，午火临月建，未土生合旺，巳火旺，辰戍丑土相，亥水，寅卯木为休囚，申酉金衰，子水月破。
未月，未土临月建，辰戍土旺，申酉金相，午火平合有气，巳火有余气，寅卯木休囚，亥子水衰，丑土月破。
申月，申金临月建，酉金旺，亥子水相，巳火平合有气，辰戍丑未土、午火休囚，卯木衰，寅木月破。
酉月，酉金临月建，申金旺，亥子水相，辰土平合有气，丑未戍土、巳午火休囚，寅木衰，卯木月破。
戍月，戍土临月建，丑未土旺，申酉金相，卯木平合有气，巳午火寅木休囚，亥子水衰，辰土月破。
亥月，亥水临月建，寅木生合旺，子水旺，卯木相，申酉金、辰戍丑未土休囚，午火衰，巳火月破。
子月，子水临月建，亥水旺，寅卯木相，丑土平合有气，申酉金、辰戍未土休囚，巳火衰，午火月破。
丑月，丑土临月建，辰戍土旺，申酉金相，巳午火、寅卯木休囚，子水克合衰，亥水衰，未土月破。
```
